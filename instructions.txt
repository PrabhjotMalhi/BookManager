Book Collection Manager - Setup Instructions

1. Install Dependencies:
   npm install

2. Start the Application:
   npm start

The application will be available at http://localhost:3000

Features:
- Add, edit, and delete books
- Filter books by genre
- Sort books by title, author, or rating
- Search books by title or author
- Bulk delete books by genre
- Responsive design for mobile devices

Note: The SQLite database will be automatically created in the db/ directory when you first run the application.
